# project-landingpage

Landing page responsif untuk perusahaan teknologi.  
Dibuat dengan: Tailwind CSS (Play CDN), HTML, sedikit JavaScript.

## Struktur folder
project-landingpage/
├── index.html
└── assets/
├── images/
├── css/
└── js/

## Menjalankan secara lokal
1. Buka `index.html` di browser, atau
2. Gunakan Live Server di VSCode.

## Deployment
Repository ini dapat di-publish via **GitHub Pages** (branch `main`, folder root).
